# Campfire cli client

A `~/.hobostove.yml` file is required. When first running `hobostove` it will help you create one.

#### ~/.hobostove.yml
    ---
    subdomain: domain
    token: apitoken
    room: Room to join

## Notifications

### Mac

Install the gem `terminal-notifier` and Mountain Lion notifications will work.

    $ gem install termina-notifier
